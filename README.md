# Rakuten Multimodal Product Classification

This repository contains a data science and machine learning project for **product type classification** using the Rakuten France e-commerce dataset.

The goal of the project is to classify products into their correct product categories by using different types of information:

- product title and description
- product images
- multimodal models combining both text and image data

The project compares text-based, image-based, and multimodal approaches in order to evaluate how much each data modality contributes to the final classification performance.

---

## Project Overview

E-commerce platforms contain large product catalogs with heterogeneous information such as product names, descriptions, images, and category labels.  
Manual product classification is time-consuming and difficult to scale.

In this project, we build and compare machine learning models that automatically predict the product type of each item.

The project includes:

- exploratory data analysis
- text preprocessing and feature engineering
- image preprocessing and augmentation
- classical machine learning models

## Dataset

The project uses the **Rakuten France Multimodal Product Data Classification** dataset.

The dataset contains product information such as:

- product ID
- product title
- product description
- product image
- product type code

Raw data and images are not stored directly in this Git repository.  
They are stored locally under the `data/raw/` directory after setup.

Dataset hosting and sharing are handled through Kaggle.
